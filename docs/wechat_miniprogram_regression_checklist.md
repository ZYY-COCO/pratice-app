# 微信小程序回归测试清单

本清单用于港研通微信小程序每次预览、体验版或正式发布前的回归测试。目标是确认登录、首页、刷题、错题、会员、公告和 AI 助教等核心链路在小程序环境可用。

## 测试前准备

- 构建命令：`cd frontend && npm run build:mp-weixin`
- 微信开发者工具导入目录：`frontend/dist/build/mp-weixin`
- 后端健康检查：`https://www.gangyantong.com/api/health`
- 期望健康检查结果：`{"status":"ok"}`
- 微信小程序后台需要配置 request 合法域名：`https://www.gangyantong.com`
- 微信小程序后台需要配置 uploadFile 合法域名：`https://www.gangyantong.com`
- 后端已配置 `WECHAT_MINIPROGRAM_APP_ID`、`WECHAT_MINIPROGRAM_APP_SECRET` 和 `WECHAT_AUTH_PASSWORD_SECRET`
- 微信公众平台已完成《用户隐私保护指引》并声明头像 / 文件选择用途

- 测试日期：

- 测试设备：
- 测试账号：
- 当前分支 / commit：
- 测试结果：通过 / 未通过

## P0 冒烟测试

以下项目必须全部通过，否则不建议继续发布。

- 小程序可以打开首页或登录页，无白屏。
- 控制台没有影响主流程的红色运行时报错。
- 微信快捷登录成功。
- 邮箱备用登录成功。
- 登录后首页数据可以加载。
- 可以进入中华文化、英语运用、数学基础任一模块。
- 可以进入刷题范围页并开始刷题。
- 可以选择选项并提交答案。
- 提交后可以看到正确答案和解析。
- 退出再进入后登录状态保持正常。

## 账号与登录

- 邮箱登录：正确账号密码可以登录。
- 邮箱登录：错误密码有明确提示，不进入首页。
- 密码输入框小眼睛可以切换显示 / 隐藏密码。
- 手机号入口显示为暂不可用或即将开放，不误导用户。
- 小程序点击微信快捷登录后能取得 code，并创建或登录同一 openid 对应的用户。
- 微信登录失败时只显示一次明确提示，不重复跳转。
- access token 临近过期时可以使用 refresh token 自动续期。
- refresh token 失效时只跳转一次登录页，不产生连续 401 或页面栈叠加。
- 新用户注册后自动获得 1 个月会员。
- 登录过期后可以重新登录，不出现卡死或空白页。
- 点击《用户隐私保护指引》可以打开微信平台配置的指引。

## 首页与公告

- 首页顶部用户信息显示正常。
- 本周刷题、总正确率、错题数显示正常。
- 中华文化、英语运用、数学基础入口可以点击。
- 公告铃铛入口显示正常。
- 新公告首次登录会弹出。
- 用户关闭公告后，本次已读公告不再反复弹出。
- 点击铃铛可以重新查看公告。
- 普通用户不会看到后台管理入口。
- 管理员账号可以看到后台管理入口。

## 刷题范围页

中华文化：

- 学习范围显示总题量和已学习题量。
- 专项刷题可以选择一个或多个考点。
- 综合刷题可以直接开始。
- 本轮题量滑块可调整。
- 开始刷题按钮状态正确。

英语运用：

- 语言知识模块可以加载题量。
- 词汇、语法、语用分类显示正常。
- 英文题干和选项不溢出。
- 解析中文显示清楚。

数学基础：

- 数学公式在题干、选项、解析中可读。
- 分数、极限、积分、偏导等符号不显示为原始 LaTeX 字符串。
- 长公式不会把页面撑破。
- 选项高度和对齐正常。

逻辑推理：

- 长题干可以正常换行。
- 选项不被遮挡。
- “练习中”“第几题中”等无关来源字样不应出现在题干里。

## 刷题流程

- 第一题按钮状态正确。
- 上一题按钮可以返回上一题。
- 未选择答案时提交按钮不可用或有提示。
- 选择答案后可以提交。
- 答对、答错样式区分清楚。
- 解析展开后显示正确答案。
- 收藏按钮可用。
- “不熟悉，加入复习”可以直接进入下一题，并计入复习 / 错题逻辑。
- 做完本轮后可以进入结果页或返回刷题范围。
- 返回刷题范围后统计数据刷新正常。

## 错题与复习

- 答错题会进入错题本。
- 不熟悉题会进入复习列表。
- 开始复习可以加载待复习题。
- 复习提交后更新复习状态。
- 错题详情可以看到题干、选项、答案和解析。
- 清除或完成复习后数量变化正常。

## AI 助教

- 右侧圆形 `AI` 悬浮入口显示正常。
- 悬浮入口可以拖动，并自动贴靠左右两侧。
- 点击入口可以打开底部浮窗。
- 浮窗不会全屏遮住题目。
- 关闭按钮易点击，关闭后返回原刷题页。
- 未提交时推荐问题不泄露答案。
- 已提交后可以解释正确答案和干扰项。
- 发送问题后聊天区自动滚动到底部。
- 请求失败时显示友好提示。
- 数学题回复中的公式尽量保持可读。

## 会员与权限

- 新注册用户显示 1 个月会员。
- 会员状态在个人中心显示正确。
- 非会员受限功能提示清楚。
- 后台授权会员后，用户重新进入可以看到会员状态变化。
- 后台取消会员后，用户重新进入可以看到状态变化。

## 后台管理

仅管理员账号测试：

- 后台管理入口显示在“其他服务”中。
- 用户列表可以加载。
- 可以搜索用户邮箱。
- 可以查看用户刷题量和会员状态。
- 可以开通 1、3、6、12 个月会员。
- 可以取消会员。
- 反馈列表可以查看和处理。
- 题库列表可以按科目 / 模块筛选。
- 题目详情可以查看题干、选项、答案和解析。
- 题目前端显示“下架”，底层使用 `archived` 状态，不直接硬删除。
- 消息管理可以新增、编辑、发布公告。

## 兼容性检查

- iPhone 模拟器显示正常。
- Android 模拟器显示正常。
- 小屏设备底部按钮不被系统栏遮挡。
- 长文本、长英文、长公式不溢出。
- 网络慢时有加载状态。
- 接口失败时有错误提示。
- 主要页面没有明显卡顿。

## 发布阻断问题

出现以下任一问题，应暂停发布：

- 无法登录。
- 首页白屏。
- 题库无法加载。
- 无法开始刷题。
- 无法提交答案。
- 答案或解析明显错乱。
- 数学公式大面积不可读。
- 普通用户可以进入后台管理。
- 会员权限错误。
- 公告无法关闭或反复弹出。
- 控制台持续出现影响主流程的红色错误。
