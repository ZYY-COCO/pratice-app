# 2026 中华文化真题命题特征分析

依据用户提供的 2026 年中华文化真题截图整理。

## 1. 题型特征

- 以单项选择题为主
- 大量题目为“人物 / 典籍 / 制度 / 术语 / 器物 / 建筑”直接识别型
- 少量题目采用“以下哪项不正确”的逆向判断方式
- 出现基于短文阅读的 2-3 题小组题

## 2. 覆盖范围特征

本套真题覆盖面非常广，明显不是只考单一传统知识点，而是跨模块抽取：

- 哲学学派与代表人物
- 历史制度与史学典籍
- 文学社团、诗歌风格、作家作品
- 建筑、绘画、音乐、陶瓷
- 水利、医学、地理、文字等科技文化常识

这说明后续 AI 生成题库时，要避免只在某一个子模块内反复出题，而要保持“广覆盖 + 高频核心点”的风格。

## 3. 命题风格特征

### 3.1 高频考法

- “某人物 / 某书 / 某制度 / 某术语 属于什么”
- “以下哪项正确 / 不正确”
- “哪一项与其他项不同”
- “某句名言、某首诗、某个典故 对应哪位人物 / 哪种思想”

### 3.2 干扰项特点

- 选项长度普遍接近
- 同类项混淆多
- 干扰项通常不是胡乱编造，而是相邻知识点

例如：

- 同一时代不同人物混淆
- 同类典籍混淆
- 同一领域不同制度、不同器物混淆

### 3.3 难度特点

- 整体偏“识记 + 准确辨认”
- 少数题要求跨知识点联想
- 不是深学术论述题，而是广度型、稳定型常识题

## 4. 对后续 AI 批量出题的启发

后续生成中华文化题时，建议优先遵守以下策略：

1. 每一批题不要只围绕一个超窄知识点打转
2. 优先考核心人物、核心典籍、核心制度、核心器物
3. 多出“辨识型”题，不要过多生成长篇分析题
4. 干扰项尽量从同一时代、同一流派、同一门类中挑选
5. 可适当加入“不正确的是”“不同的是”等真题常见问法
6. 阅读理解型文化题可单独作为第二阶段生成，不必一开始大批量铺开
